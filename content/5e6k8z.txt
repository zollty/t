<p>题干=</p><p>电信的1M宽带，理论上下载速度最快是多少KB/s？下载1MB的数据需要多少秒？</p><p>答案=</p><p>128 KBps（KB/s），8秒</p><p>分析=</p><p>电信部门说的1M宽带的M是指Mb/s，下载软件时的传输率是MB/S，MB/s和Mb/s是有区别的。</p><p>其中<span style="color: rgb(192, 0, 0);"><strong>大B代表</strong></span>Byte(<span style="color: rgb(192, 0, 0);"><strong>字节</strong></span>)，<span style="color: rgb(192, 0, 0);"><strong>小b</strong><strong>代表</strong></span>bit(比特 或 <span style="color: rgb(192, 0, 0);"><strong>位</strong></span>)。</p><p><strong>1 Byte = 8 bit</strong></p><p><strong>1 KB = 1024 Byte</strong></p><p><strong>1 Kb = 1024 bit</strong></p><p>1 Mb = 1024 Kb</p><p>1 MB = 1024 KB</p><p>1 MB = 8Mb</p><p><strong>1 Mb = 0.125 MB</strong></p><p><span style="color: rgb(255, 0, 0);">注意：上面没有考虑1000进制的单位，</span></p><p><span style="color: rgb(255, 0, 0);">有些地方用的1000进制单位，那么</span></p><p><span style="color: rgb(255, 0, 0);">1 KB = 1000 B</span></p><p><span style="color: rgb(255, 0, 0);">1 KiB = 1024 B</span></p><p><span style="color: rgb(255, 0, 0);">1 MB = 1000 KB</span></p><p><span style="color: rgb(255, 0, 0);">1 MiB = 1024 KiB</span></p><p><br/></p><p>硬盘、U盘就是采用的1000进制单位：</p><p>1GB=1000MB&nbsp;</p><p>1MB=1000KB&nbsp;</p><p>1Kb =1000B</p><p>因此我们常常会发现买回来的硬盘没有商家的所宣传那么大。</p><p>比如1T的硬盘，按我们的想法应该是1024GB大小。<br/></p><p>但实际只有930GB左右大小。<br/></p><p>因为，1T的硬盘在商家眼中的大小是 1000 000 000 000 Byte<br/></p><p>我们可以换算一下：1000 000 000 000 / 1024/1024/1024 = 931.3225 GiB<br/></p><p><br/></p><p>在数据传输率上官方数据中(如电信部门)一般采用Mb/s为单位，即，单位是比特（bit）。</p><p>而下载软件(如IE、迅雷、快车)一般采用MB/s为单位，我们所说的文件10M也是指的MB，即，单位是字节（Byte）。</p><p>宽带最高下载理论值（宽带值除以8）：</p><p><strong><span style="color: rgb(192, 0, 0);">1Mb/s = 0.125MB/s = 128KB/s</span></strong></p><p>按这个说法1M的宽带最快下载速度是0.125MB/s（128KB/s），10M的宽带最快下载速度是1.25MB/s，100M的宽带最快下载速度是12.5MB/s。</p><p><br/></p><p>1M的宽带下载1MB数据需要8秒，下载10MB需要80秒，一般一个网站500KB~2MB，1M的宽带打开需要10秒左右，会比较慢。</p><p><br/></p>
