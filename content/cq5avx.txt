<p>题干=</p><p>为了提高文件拷贝的效率，通常在IO操作的时候怎么做缓存？</p><p>答案=</p><p>1.使用buffer数组缓存，然后将数据存入buffer，例如：while(in.read(buffer) != -1) {}；</p><p>2.使用BufferedReader和BufferedWriter。</p><p><br/></p><p>题干=</p><p>给定一个文件路径，如何打开文件流，读取里面的字符内容？</p><p>答案=</p><p>通过<strong>字节流----&gt;字符流----&gt;字符包装流</strong>。</p><p>&nbsp;File file = new File(“src/res/abc.txt”);//读取路径</p><p>&nbsp;FileInputStream fis = new FileInputStream(file); //字节流</p><p>&nbsp;InputStreamReader is = new InputStreamReader(fis); //字符流</p><p>&nbsp;BufferedReader br = new BufferedReader(is); // 包装流</p><p>即：new&nbsp;<strong>BufferedReader</strong>(new&nbsp;<strong>InputStreamReader</strong>(new&nbsp;<strong>FileInputStream</strong>(&quot;C:/java/hello&quot;)))&nbsp;</p><p><br/></p>
