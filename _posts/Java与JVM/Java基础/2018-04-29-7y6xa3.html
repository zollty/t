题干=

什么是 封装、多态，重载和重写？

答案=

  • 封装：隐藏属性、方法或实现细节的过程称为封装。

  • 多态：同一个实现接口，使用不同的实例而执行不同操作。多态性是不同的实例对象以不同的方式对相同的信息作出不同的表现。

  • 重载：发生在同一个类中，方法名必须相同，参数类型不同、个数不同、顺序不同，方法返回值和访问修饰符可以不同，发生在编译时。（只有方法名相同）

  • 重写：发生在父子类中，方法名、参数列表必须相同，返回值小于等于父类，抛出的异常小于等于父类，访问修饰符大于等于父类；如果父类方法访问修饰符为private则子类中就不是重写。
