题干=

为了提高文件拷贝的效率，通常在IO操作的时候怎么做缓存？

答案=

1.使用buffer数组缓存，然后将数据存入buffer，例如：while(in.read(buffer) != -1) {}；

2.使用BufferedReader和BufferedWriter。


题干=

给定一个文件路径，如何打开文件流，读取里面的字符内容？

答案=

通过字节流---->字符流---->字符包装流。

 File file = new File(“src/res/abc.txt”);//读取路径

 FileInputStream fis = new FileInputStream(file); //字节流

 InputStreamReader is = new InputStreamReader(fis); //字符流

 BufferedReader br = new BufferedReader(is); // 包装流

即：new BufferedReader(new InputStreamReader(new FileInputStream("C:/java/hello")))
