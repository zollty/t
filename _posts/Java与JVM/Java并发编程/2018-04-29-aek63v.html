题干=

Java并发编程里面有哪些常用的API？至少说出3种

答案=

  • 集合：ConcurrentMap、CopyOnWriteArrayList、BlockingQueue等

  • 原子数据：AtomicLong、AtomicBoolean等

  • 锁：ReentrantLock、ReentrantReadWriteLock（tryLock,unlock）

  • 同步：CountDownLatch（countDown,wait）

  • 线程：ExecutorService，ScheduleExecutorService，Executors（newFixedThreadPool, newCachedThreadPool, newSingleThreadExecutor，newScheduledThreadPool ）
