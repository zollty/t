题干=

Linux中，查找/tmp/sure目录下，且包含password字符的所有文件，并删除它们。

答案=

grep -lr "password" /tmp/sure/ | xargs rm -rf


题干=

Linux，test.txt文件中有多条记录，找出这个文件中没有重复行的记录。

答案=

查找非重复行

sort file |uniq -u

查找重复行

sort file |uniq -d

去除重复行

sort file |uniq

统计

sort file | uniq -c
